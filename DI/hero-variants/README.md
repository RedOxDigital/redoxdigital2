# Hero Section Variants

Four unique hero section designs for Digital Marketing Strategy Consultant, following mobile-first design principles.

## Variant 1: Stacked Centered
- **Layout**: Mobile-first stacked approach with centered headline and value proposition
- **Typography**: Large, bold centered headline with left-aligned body text
- **Visual Treatment**: Clean gradient background with circular icon features
- **Best For**: Clean, professional presentation with strong visual hierarchy

## Variant 2: Split Screen
- **Layout**: Side-by-side on desktop, stacked on mobile (text left, visual right)
- **Typography**: Edge-anchored large headline with inline value proposition
- **Visual Treatment**: Abstract gradient pattern with geometric shapes
- **Best For**: Modern, dynamic presentation with visual interest

## Variant 3: Full Bleed
- **Layout**: Full-width background with overlay and edge-anchored content
- **Typography**: Extra-large bold headline with high contrast white text
- **Visual Treatment**: Dark gradient background with pattern overlay
- **Best For**: Bold, immersive experience with maximum impact

## Variant 4: Card Based
- **Layout**: Modular card system with main content card and feature grid
- **Typography**: Structured card-based typography with clear hierarchy
- **Visual Treatment**: Clean white cards on light background with shadows
- **Best For**: Scannable, organized content with modular flexibility

## Design Compliance

All variants follow the Mobile-First Design Mandate:
- ✅ Body text ≥16px
- ✅ Left-aligned body text
- ✅ High contrast (≥4.5:1)
- ✅ Line height ≈1.5x
- ✅ Paragraphs ≤3 lines
- ✅ Mobile-first responsive design
- ✅ Single primary CTA per variant
